📌 Overview
EduTrack is a Java Swing-based desktop application designed to simplify student and teacher management.
It provides an intuitive GUI where teachers can manage academic data, and students can track their performance.

✨ Features
👩‍🏫 Teacher Panel
➕ Add & manage students

📅 Update attendance

📝 Assign marks & grades

📢 Manage academic events

👨‍🎓 Student Panel
👤 View profile

📊 Check attendance percentage

🏆 View marks & grades

🔔 Get alerts for low attendance

⚙️ System Features
🔐 Secure login system

⚠️ Automatic alert (<75% attendance)

📈 Result calculation

💾 Data storage (MySQL)

🛠️ Tech Stack
Layer	Technology
